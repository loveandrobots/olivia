# Tools

## Documentation Tools

- **Spec template**: `docs/specs/spec-template.md` -- standard template for all feature specs. Includes sections for workflow, system behavior, trust model, acceptance criteria, and open questions.
- **Decision history**: `docs/learnings/decision-history.md` -- append decisions with rationale after spec work.
- **Assumptions log**: `docs/learnings/assumptions-log.md` -- track working beliefs with confidence levels.
- **Glossary**: `docs/glossary.md` -- canonical product terms. Reference before introducing new terminology.
- **Milestones**: `docs/roadmap/milestones.md` -- current milestone status and gate requirements.

## Paperclip Skills

- **paperclip**: Coordination with the control plane -- check assignments, update status, comment, delegate.
- **olivia-spec**: Feature spec writer for Olivia. Use when drafting new specs.
- **olivia-log**: Learning system updater. Use when updating decision history or assumptions.
- **olivia-review**: Doc review. Use when reviewing documentation quality.
- **olivia-orient**: Session orientation. Use at the start of product work sessions.
