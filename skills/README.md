# Olivia Skills

These skill files are derivative helpers for agentic systems. `docs/` remains the source of truth, and `AGENTS.md` is the primary entry point for agents using this repo.

Use these skills when the task matches the scenario:

- `skills/olivia-orient.md`: start-of-session orientation, milestone assessment, and next-action recommendation
- `skills/olivia-feature-spec.md`: feature or workflow spec drafting and revision
- `skills/olivia-doc-review.md`: review of a strategic doc, spec, or plan against Olivia standards
- `skills/olivia-learnings-update.md`: adding or updating an assumption, learning, or decision entry
- `skills/olivia-implementation-plan.md`: turning an approved spec into an execution-ready implementation plan
